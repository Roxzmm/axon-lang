# Axon Calculator

Simple CLI calculator built with Axon language.

## Usage

```bash
axon run apps/calculator/calculator.axon
```

## Features

- Basic arithmetic: `+`, `-`, `*`, `/`, `%`
- Power operator: `^`
- Complex expressions with parentheses
- Variables and constants

## Example

```
Axon Calculator
===============

Basic operations:
10 + 5 = 15
10 - 5 = 5
10 * 5 = 50
10 / 5 = 2
10 % 3 = 1

Power (2^8):
2 ^ 8 = 256

Complex expressions:
10 + 5 * 2 = 20
(10 + 5) * 2 = 30
100 / 3 = 33

Calculator demo complete!
```
