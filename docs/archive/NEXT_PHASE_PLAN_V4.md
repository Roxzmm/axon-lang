## 技术架构

### 字节码格式
```
+------------------+
| Header (magic)   |
+------------------+
| Constant Pool    |
+------------------+
| Function Table   |
+------------------+
| Code             |
+------------------+
```

### 自举流程
```
Step 1: TS 解释器
  main.ts (TS) → 运行 → axon 程序

Step 2: TS 编译 Axon 编译器
  main.ts (TS) → 编译 → compiler.axon (Axon源码)

Step 3: 用 TS 解释器编译 Axon 源码
  main.ts (TS) + compiler.axon → 编译 → compiler.axbc (字节码)

Step 4: 用 VM 运行字节码编译器
  VM + compiler.axbc → 编译 → user_program.axbc

Step 5: 自举完成！
  compiler.axbc + VM → 编译 → compiler.axbc (自己编译自己)
```
